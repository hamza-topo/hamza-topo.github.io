Introduction:

Puppeteer - the secret agent of the web! With it, you can control headless Chrome and retrieve video sources from Instagram Reels links. Let's dive in and see how to extract the video goodness from these links using the power of Puppeteer.

Setting up the environment:

First, create a new folder for your project. Then, in the terminal, navigate to that folder and run the following command to install Puppeteer: npm install puppeteer

Let's code:

Simply create an empty file named app.js in the same folder where you ran the npm install puppeteer command. You are now ready to start writing code in the app.js file.

  • Import the puppeteer and dotenv libraries.
  • Set a variable link1 to the URL of an Instagram Reel.
  • Define an asynchronous function spider that takes a videoUrl as a parameter.
  • Launch a headless browser using puppeteer.
  • Create a new page in the browser.
  • Navigate the page to the URL stored in the INSTAGRAM_URL environment variable and wait until the network is idle.
  • Type the username and password stored in the INSTA_USERNAME and INSTA_PASSWORD environment variables into the username and password fields on the page.
  • Click the submit button on the page to log in.
  • Wait for the page to navigate to a different URL after logging in.
  • Wait for an element with the class ._acan to be visible on the page.
  • Click the button element within a div element.
  • Navigate the page to the videoUrl passed as a parameter to the spider function and wait until the network is idle.
  • Retrieve the src attribute of the video element on the page using the evaluate() method of puppeteer.
  • Log the video source to the console.
  • Return the video source.
  • Call the spider function and pass the link1 variable as a parameter.

To run the script, open a terminal/command prompt and navigate to the directory where the script file (app.js) is located. Then, type "node app.js" and press Enter. The script will be executed and the output will be displayed in the terminal/command prompt.

Conclusion:

In conclusion, the script demonstrates the usage of Puppeteer, a popular Node.js library, to scrape the source of a video from an Instagram Reel. It imports the required libraries, sets variables, creates a headless browser, logs in to Instagram, navigates to the Reel's page, retrieves the video source, logs it to the console, and returns it. Running the script using "node app.js" will execute the code and display the video source in the console.
